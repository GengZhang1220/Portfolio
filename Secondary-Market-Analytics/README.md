# Secondary Market Assets Real-time Analytics

Project Overview:
Developed an end-to-end Robotic Process Automation (RPA) workflow to extract, process, and analyze real-time pricing data for secondary market commodities. This tool eliminates manual data entry and provides accurate valuation tracking.

Technical Highlights:

Automated Data Extraction: Engineered automated workflows to scrape complex pricing and inventory data from targeted secondary market platforms.

JSON Data Parsing: Successfully processed and structured deeply nested JSON responses, ensuring high-fidelity extraction of asset attributes and market values.

Data Visualization & Analytics: Integrated the cleaned data with Excel Power Query to automatically aggregate market trends and generate comprehensive financial reporting.

Business Value:
Transformed hours of manual market research into a seamless, automated process, enabling data-driven decision-making for asset valuation.
