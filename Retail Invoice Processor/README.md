# Project Overview: 
Developed a lightweight RPA workflow designed to automate the extraction of critical vendor and pricing data from retail delivery receipts and invoices. 

# The Motivation: 
Drawing from my hands-on experience managing inventory and handling supplier returns in the fast-paced grocery retail sector, I identified manual invoice reconciliation as a significant operational bottleneck. I proactively repurposed and expanded upon foundational automation concepts to build a practical tool that directly addresses this real-world inefficiency. 

# Technical Highlights: 

Data Digitization: Utilized data extraction techniques to pull specific fields (e.g., Vendor Name, Invoice Number, Total Amount) from raw document formats. 

Format Conversion: Automated the transformation of unstructured receipt data into structured, ready-to-use Excel reports for fast accounting reconciliation. 

# Business Value: 
Demonstrates the ability to identify grassroots operational pain points and rapidly prototype technological solutions to reduce human error and save valuable labor hours.
